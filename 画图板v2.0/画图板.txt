Java中的注释：
单行注释
 	//注释内容，通常是给某一行代码进行简短的注释
 
多行注释
	/*
		通常是给某一行代码进行详细注释
		可以有多行
	*/
	
	/*
	 *通常是给某一行代码进行详细注释
	 *可以有多行
	 */
	 
文档注释
	/**
	 *通常是给类，构造方法，属性，方法写注释
	 *可以有多行
	 *可以根据文档注释自动生成使用说明文档
	 */

BorderLayout 边框布局

自定义颜色
	Color c = new Color(R,G,B);
	R,G,B的值是0~255之间的数字，或者0.0~1.0之间的浮点数

窗口的默认布局是ViewportLayout
面板的默认布局就是FlowLayout
